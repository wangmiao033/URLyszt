# 文件上传指南

由于路径编码问题，请选择以下任一方法上传文件到GitHub：

## 方法一：使用批处理文件（最简单）⭐

1. 双击运行 `上传.bat` 文件
2. 脚本会自动执行所有上传步骤
3. 如果遇到问题，请查看错误信息

## 方法二：使用PowerShell脚本

1. 在项目文件夹中，**右键点击空白处**
2. 选择 **"在终端中打开"** 或 **"Open in Terminal"**
3. 执行以下命令：
   ```powershell
   .\upload.ps1
   ```

## 方法三：手动执行Git命令

在PowerShell或Git Bash中，**手动进入项目目录**后执行：

```powershell
# 1. 进入项目目录（请根据实际路径调整）
cd "H:\Y云上征途ios\URL"

# 2. 配置Git安全目录
git config --global --add safe.directory "H:/Y云上征途ios/URL"

# 3. 添加文件
git add index.html vercel.json upload.ps1

# 4. 提交更改
git commit -m "添加index.html和vercel.json配置"

# 5. 推送到GitHub
git push origin main
```

如果main分支不存在，使用：
```powershell
git push origin master
```

## 方法四：使用GitHub网页上传（推荐，最简单）⭐

如果命令行遇到问题，可以直接在GitHub网页上传：

1. **访问仓库：** https://github.com/wangmiao033/URLyszt
2. **点击 "Add file" 按钮** → 选择 **"Upload files"**
3. **拖拽以下文件**到上传区域：
   - `index.html`
   - `vercel.json`
   - `upload.ps1`（可选）
4. **在页面底部**输入提交信息：`添加index.html和vercel.json配置`
5. **点击 "Commit changes"** 按钮

## 需要上传的文件

- ✅ `index.html` - 隐私政策主页（重要！）
- ✅ `vercel.json` - Vercel配置文件（重要！）
- ⚠️ `upload.ps1` - 上传脚本（可选）

## 上传后验证

上传成功后，访问以下URL确认：

- **隐私政策URL：** https://ur-lyszt.vercel.app/
- **GitHub仓库：** https://github.com/wangmiao033/URLyszt

如果Vercel已连接GitHub，上传后会自动重新部署。

## 常见问题

### Q: 提示 "dubious ownership" 错误？
A: 执行：`git config --global --add safe.directory "H:/Y云上征途ios/URL"`

### Q: 提示 "permission denied"？
A: 检查SSH密钥是否已配置，或使用HTTPS方式：
   ```powershell
   git remote set-url origin https://github.com/wangmiao033/URLyszt.git
   ```

### Q: 找不到文件？
A: 确保在正确的项目目录中执行命令
